### 页面中的省略号发生了什么？

数行文字，日增其一，永不省略，则流量汹于洪水，体积重于泰山。

### 为什么不用服务器续？

人呐就都不知道，开发机就可以续。续命的脚本啊，当然要靠自己运行，但是也要考虑到其它的进程。

### 为什么不用 Python？

Python 语我知道，写机器学习很好。老先生也支持。

### 我也想续，该怎么做？

Fork 这个仓库并克隆到本地，然后定时或手动运行`commit.sh`，坚持不懈，日积月累，就一定能不断有所收获。
